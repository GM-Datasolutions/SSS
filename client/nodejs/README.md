# DICOM NodeJS Client

Work in progress

## Getting Started

Built using `lts/carbon` (8.x as of this writing)

```
npm install
```

## Usage

Update `examples/index.js` to change host/port as appropriate.

```
$ node examples/index.js

{ method: 'ping', ping: 'pong' }
{ method: 'echo', echo: 'hello, world!' }
```
